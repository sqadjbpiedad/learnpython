**Build a foundation for Python excellence.**

**Essentials** is delivered as a 16-hour introductory course that provides a comprehensive foundation of the fundamental knowledge and practical skills that you need in order to use Python.

## Who is it for?
Those who are **new to programming** and/or current programmers who **looking to upskill themselves with Python**.

## Learning Objectives
At the end of this module, you should be able to:

1. Install and manage Python environments and packages.
2. Declare and manipulate variables, understand data types, and explain scope in Python.
3. Implement and use various data structures in Python.
4. Understand and work with strings and ranges in Python.
5. Write code that uses flow control statements (loops and conditionals) to make decisions and iterate through data.
6. Use iterations and comprehensions to simplify code and improve performance.
7. Define, create, and utilize functions, classes, and objects in Python.
8. Read and write user input and files using Python's built-in functions.
9. Handle errors and exceptions effectively in Python.
10. Apply lambda, map, and filter functions to solve problems efficiently.

## Coverage
- Environment and package management
- Variables, data types, and scope
- Data structures
- Strings and ranges
- Flow control (loops and conditionals)
- Iterations and comprehensions
- Functions, classes, and objects
- User inputs and file handling
- Errors and exceptions
- Lambda, map, and filter functions
- Basic visualizations
