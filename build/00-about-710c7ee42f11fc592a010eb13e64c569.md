**Master the building blocks of data work in Python.**

**Arrays and DataFrames** is delivered as a 16-hour introductory course that provides hands-on experience with the core tools for data analysis and scientific computing in Python. Whether you're new to data work or looking to deepen your skills, this module will equip you with the ability to structure, process, and analyze data effectively.

## Who is it for?
- Data analysts and scientists looking to enhance their Python skills.
- Students and professionals new to data analysis.
- Anyone interested in data manipulation and preparation for machine learning projects.
- Developers aiming to integrate data processing into their applications.

## Learing Objectives
At the end of this module, you should be able to:

1. Understand the fundamentals of arrays and their applications in numerical computations.
2. Master the creation, manipulation, and analysis of DataFrames for data processing.
3. Perform data cleaning, transformation, and aggregation using Python libraries.
4. Apply techniques for handling missing data, merging datasets, and data visualization.
5. Utilize arrays and DataFrames to solve real-world data analysis problems.

## Coverage

- Introduction to arrays with numpy
- Introduction to DataFrames with pandas and polars
- Data manipulation and analysis
- Advanced techniques and applications
